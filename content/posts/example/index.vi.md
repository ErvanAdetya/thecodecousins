### Phần 1

Một đoạn văn cho có

##### Mục nhỏ 1

Một đoạn văn cho có cho mục nhỏ 1

##### Thử code highlight

Code Go dùng hệ thống highlight của hugo

{{< highlight go >}}
a := "some string"
fmt.Println("lol")
{{</ highlight >}}

Code Go dùng 3 backticks

```go
a := "some string"
fmt.Println("lol")
```

Code JS dùng hệ thống highlight của hugo

{{< highlight js >}}
const a = "some string"
console.log("lol")
const b = () => { return 1; };
{{</ highlight >}}

Code JS dùng 3 backticks

```js
const a = 'some string';
console.log('lol');
const b = () => {
  return 1;
};
```

Tốt nhất là nên dùng 3 backticks trừ khi cần đánh số dòng
