##### Thử code highlight

{{< highlight go >}}
a := "some string"
fmt.Println("lol")
{{</ highlight >}}

```go
a := "some string"

```

{{< highlight js >}}
const a = "some string"
console.log("lol")
const b = () => { return 1; };
{{</ highlight >}}

```js
const a = 'some string';
console.log('lol');
const b = () => {
  return 1;
};
```

Tốt nhất là nên dùng 3 backticks trừ khi cần đánh số dòng
